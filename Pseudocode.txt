Laboratory Activity 1: Add Two Numbers
Pseudocode:
START
INPUT num1
INPUT num2
PROCESS sum = num1 + num2
DISPLAY sum
END

Laboratory Activity 2: Compute the average of Three Quiz Scores
Pseudocode:
START
INPUT Q.score1
INPUT Q.score2
INPUT Q.score3
PROCESS score = score1 + score2 + score3
Display score
End


Laboratory Activity 3: Determine if a Number is Even or Odd
Pseudocode:
START 
INPUT number 
IF number MOD 2 = 0 THEN   
   Display "Even"
ELSE   
   Display  "Odd"
END IF
END
